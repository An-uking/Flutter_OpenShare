# Flutter_OpenShare Plugin
Flutter_OpenShare是一款集成OpenShare自动获取邀请码或参数的flutter 插件,可以帮助开发者快速集成

### 推广渠道
    1.邀请码
    2.软文推广
    3.广告营销
    4.商品分销
注:可以自定义推广渠道，上面几个只是简单举例
